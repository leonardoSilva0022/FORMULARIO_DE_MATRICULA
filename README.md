# 🌟 Formulário de Matrícula — Estrelas do Amanhã

Projeto de formulário web para matrícula de crianças na escola de educação infantil **Estrelas do Amanhã**. Desenvolvido com HTML e CSS puros, com foco em uma interface amigável, acessível e visualmente agradável.

🔗 **Demo ao vivo:** [leonardosilva0022.github.io/FORMULARIO_DE_MATRICULA](https://leonardosilva0022.github.io/FORMULARIO_DE_MATRICULA/)

---

## 📋 Sobre o projeto

O formulário guia os responsáveis pelo preenchimento de todas as informações necessárias para a matrícula, organizadas em seções claras e objetivas. O layout é dividido em duas colunas: o formulário à esquerda e um painel de apresentação da escola à direita.

---

## ✨ Funcionalidades

- **Informações da criança** — nome completo, data de nascimento, sexo, informações médicas e upload da certidão de nascimento
- **Endereço residencial** — CEP com preenchimento automático dos campos de rua, cidade e estado
- **Informações do responsável** — nome, telefone e e-mail com validação
- **Opções de matrícula** — seleção de turno (Manhã ou Tarde) e modalidade esportiva (Futebol, Basquete, Natação, Yoga, Volley ou Boxe)
- **Aceite de Termos e Condições** e Política de Privacidades
- Botões de **Salvar respostas** e **Fazer matrícula**

## 🚀 Como executar localmente

   ```bash

   ```

   ```bash
   cd FORMULARIO_DE_MATRICULA
   ```

3. Abra o arquivo `index.html` no seu navegador.
